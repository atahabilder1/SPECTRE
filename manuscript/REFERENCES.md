# SPECTRE Bibliography

## Core References

Download BibTeX files from the following sources:

### 1. Ethereum Yellow Paper
**Wood, G. (2014). "Ethereum: A Secure Decentralised Generalised Transaction Ledger."**
- URL: https://ethereum.github.io/yellowpaper/paper.pdf
- Canonical EVM specification

### 2. KEVM: A Complete Formal Semantics of the Ethereum Virtual Machine
**Hildenbrandt, E., et al. (2018). IEEE CSF.**
- DOI: 10.1109/CSF.2018.00022
- URL: https://www.ideals.illinois.edu/items/98139
- Formal semantics in K framework

### 3. Making Smart Contracts Smarter
**Luu, L., et al. (2016). ACM CCS.**
- DOI: 10.1145/2976749.2978309
- Security vulnerabilities in smart contracts

### 4. An Analysis and Survey of the Development of Mutation Testing
**Jia, Y. and Harman, M. (2011). IEEE TSE.**
- DOI: 10.1109/TSE.2010.62
- Comprehensive mutation testing survey

### 5. Differential Testing for Software
**McKeeman, W. (1998). Digital Technical Journal.**
- URL: https://www.cs.dartmouth.edu/~mckeeman/references/DifferentialTestingForSoftware.pdf
- Foundation of differential testing

### 6. Simplifying and Isolating Failure-Inducing Input
**Zeller, A. and Hildebrandt, R. (2002). IEEE TSE.**
- DOI: 10.1109/32.988498
- Delta debugging algorithm
